# TallerIS — Taller de Ingeniería de Software

## Sistema de Autenticación y Carga de Datos

Aplicación de consola (CMD) que permite la autenticación de usuarios mediante múltiples fuentes (XML, Base de Datos, Web Service) utilizando el **patrón Strategy**, y la carga de información desde archivos CSV con persistencia en base de datos y notificación por correo electrónico.
